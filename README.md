# Smashlings

A Vampire Survivors-style 3D browser game built with [Three.js](https://threejs.org/), [bitecs](https://github.com/NateTheGreatt/bitECS), and TypeScript.

## Play

[https://isirux.github.io/Smashlings/](https://isirux.github.io/Smashlings/)

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move |
| Space | Jump |
| Shift | Dash |

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
